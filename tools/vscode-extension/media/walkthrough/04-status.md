# Read the status

Click a stage row to **open its artifact** — `spec.md`, `integration-ir.yaml`,
`plan.md`, the `contracts/` folder, and so on.

When a stage is **blocked** or has **findings**, an **Open Report** action jumps
straight to the report that explains why: `review-report.md`,
`contract-lint-report.json`, `*-test-report.json`, `plan-blocked.md`, etc.

Stale stages show the upstream edit that triggered them inline
(`stale · spec.md edited after integration-ir.yaml`), and the integration tooltip
gives the full findings breakdown.
