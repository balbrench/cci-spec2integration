# Compose a command

Run **Spec2Integration: Compose Command…** and pick any slash command. A guided
wizard walks you through that command's options — mode, input, flags, the target
folder — and assembles the exact `/command …` string.

The command is then handed to your live Claude Code chat via a deep link (or
copied to the clipboard). The extension never runs commands headlessly; you stay
in control of execution in chat.

> Tip: leave **Preview before send** on to review or tweak the command first.
